# Momentum — client

Next.js 14 (App Router) frontend for Momentum, a command center for Gmail,
Google Calendar, Slack and GitHub. TypeScript end-to-end, Tailwind, Clerk auth,
React Flow connection canvas, dark/light theme, Poppins.

See `docs/SETUP.md` to run locally and `docs/GIT_PLAN.md` for the commit history plan.
